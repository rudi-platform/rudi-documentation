# Créer une réutilisation

La création d'une réutilisation se fait depuis le catalogue des réutilisations, en cliquant sur le bouton "**Déclarer une réutilisation**". 

Pour créer une réutilisation, vous devez renseigner différentes informations :
- Informations décrivant le projet 
- Informations sur votre identité : vous pouvez soumettre un projet en votre nom ou au nom d'une organisation
- Sélection des jeux de données que vous souhaitez utiliser dans le cadre de votre réutilisation

La description de votre réutilisation vous permet de choisir si celle-ci sera visible depuis le catalogue ou si elle doit être confidentielle (visible uniquement par vous, l'équipe administrative Rudi et les producteurs de données concernés).

Après modération positive de votre réutilisation, vous pouvez la consulter et la modifier depuis votre compte -> Mon activité.

La modification d'une réutilisation est aussi disponible depuis cet espace si elle ne mobilise pas de jeux de données à accès restreint et est au statut technique "Validated".
